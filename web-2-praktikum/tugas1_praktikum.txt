1.Jelaskan apa yang dimaksud dengan dokumen root ?

jawaban :Dokumen root adalah direktori utama tempat file' web disimpan dan di akses oleh server web

2.Pada apache web server, apakah dokumen root harus terletak dalam direktori
  C:\xampp\htdocs ? Jelaskan jawaban anda

jawaban :Tidak, dokumen root di Apache tidak harus terletak di C:\xampp\htdocs. Lokasinya bisa diubah dengan  mengedit file httpd.conf dan mengatur DocumentRoot ke direktori lain, lalu me-restart Apache agar perubahan berlaku.

3.Jelaskan apa yang dimaksud dengan perintah git : init, clone, remote, add, commit, push
dan checkout

jawaban
-init : Membuat repository Git baru dalam direktori.
-clone : Menyalin repository yang sudah ada dari remote ke lokal.
-remote : Mengelola repository jarak jauh (remote repository).
-add : Menambahkan perubahan file ke staging area sebelum commit.
-commit : Menyimpan perubahan yang telah ditambahkan ke staging area.
-push : Mengirim commit dari repository lokal ke repository remote.
-checkout : Beralih ke branch lain atau mengembalikan file ke kondisi tertentu.

4.jelaskan perbedaan antara GitHub dan GitLab

-GitHub : Platform hosting repository Git yang populer, menawarkan fitur kolaborasi seperti pull request, issues,       dan actions. Lebih banyak digunakan untuk proyek open-source.
-GitLab : Alternatif GitHub dengan fitur lebih lengkap untuk DevOps (CI/CD bawaan). Bisa di-host sendiri (self-hosted) di server pribadi, cocok untuk perusahaan yang butuh kontrol penuh.

5.Apa bedanya variable user dengan variable system ?

-Variable User : Hanya berlaku untuk pengguna tertentu di sistem. Setiap user bisa memiliki variabel yang berbeda.
-Variable System : Berlaku untuk seluruh sistem dan semua pengguna. Biasanya digunakan untuk konfigurasi global.

6 Apa persamaan variable system dengan variable constant ?

-Digunakan untuk menyimpan nilai → Keduanya menyimpan nilai yang bisa digunakan dalam program atau sistem operasi.
-Dapat berpengaruh secara global → Variable system berlaku untuk seluruh sistem, sedangkan constant sering  digunakan dalam program sebagai nilai tetap yang bisa diakses di berbagai bagian kode.
-Membantu pengaturan dan konfigurasi → Variable system sering digunakan untuk konfigurasi sistem operasi, sementara constant digunakan dalam program untuk menyimpan nilai tetap yang tidak boleh berubah.
